### Instructions
#### 1) Clone Repository
```
git clone https://github.com/JesusOrellana/node-api-rest.git
```
#### 2) Choosing a stack
You must choose one of the two options to work with Node - Mongodb or Node - Mysql. By default the main branch contains the Node-Mongodb stack, but if you want to develop with Node-Mysql you can switch branches with the following command
```
git checkout node-api-rest-mysql
```
#### 2) Copy files
Now you just have to copy the base structure and take it to your project.
